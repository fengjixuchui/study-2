# 孤儿规则的局限性

在设计 trait 时，还需要考虑**是否会影响下游的使用者**。

比如在标准库实现一些trait时，还需要考虑**是否**需要为**所有**的 **T** 或 `&'a T` **实现该 trait**:

```rust
impl<T: Foo> Bar_3_73 for T { }
impl<'a, T: Bar_3_73> Bar_3_73 for &'a T { }
```
