
一般语言都有常用的流程控制语句: 条件语句和循环语句. Rust 中不叫流程控制语句, 而叫流程控制表达式.

# 条件表达式

表达式一定会有值, 所以 **if 表达式的分支必须返回同一个类型的值**. 所以 Rust 没有三元操作符 `? : `. if 表达式的求值规则和快表达式一致.

```rust
fn code2_16() {
	let n = 13;
	let big_n = if (n < 10 && n > -10) {
		10 * n
	} else {
		n/2
	};
	assert_eq!(big_n, 6);
}
```

变量绑定 big_n 的赋值是由一个 if 表达式来完成的.
